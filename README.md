# WebP 转 PNG/JPG 工具

这是一个用于将 WebP 图片文件转换为 PNG 或 JPG 格式的工具集，包含两种实现方式：
1. Python 命令行工具
2. 纯前端网页工具（无需 Python 环境，离线使用）

## 功能特点

### 通用功能
- 支持将 WebP 文件转换为 PNG 或 JPG 格式
- 处理 RGBA 模式的 WebP 文件（转换为 JPG 时会添加白色背景）
- 简单易用的界面

### Python 工具特点
- 支持批量转换文件夹中的所有 WebP 文件
- 自动创建输出目录
- 提供详细的转换过程输出
- 命令行交互界面

### 前端网页工具特点
- 无需安装任何软件，直接在浏览器中使用
- 支持批量选择多个 WebP 文件
- 实时预览转换结果
- 响应式设计，适配不同屏幕尺寸
- 完全离线使用，保护隐私

## 工具一：Python 命令行工具

### 环境要求
- Python 3.6+
- Pillow 库

### 安装步骤
1. 确保已安装 Python 3.6 或更高版本
2. 安装 Pillow 库：

```bash
pip install pillow
```

### 使用方法
1. 将需要转换的 WebP 文件放入 `webp` 文件夹
2. 运行转换脚本：

```bash
python webp_converter.py
```

3. 根据提示选择输出格式（png 或 jpg）
4. 转换完成后，生成的图片将保存在 `output_png` 或 `output_jpg` 文件夹中

### 示例

```
$ python webp_converter.py
请选择输出格式 (png/jpg): png
开始将 webp 文件转换为 png 格式...
成功转换: example.webp -> example.png
转换完成!
```

## 工具二：前端网页工具

### 环境要求
- 现代 web 浏览器（Chrome、Firefox、Edge 等）
- 无需安装任何软件
- 无需联网

### 使用方法
1. 在浏览器中打开 `webp_converter.html` 文件
2. 点击 "选择WebP文件" 按钮，选择一个或多个 WebP 文件
3. 根据需要点击 "转换为 PNG" 或 "转换为 JPG" 按钮
4. 转换完成后，页面会显示每个文件的预览
5. 点击图片下方的下载按钮即可保存转换后的文件

### 界面说明
- **文件选择区域**：用于选择需要转换的 WebP 文件
- **文件列表**：显示已选择的文件及其大小
- **转换按钮**：点击后直接开始转换为对应格式
- **转换结果区域**：显示转换后的图片预览和下载按钮

## 注意事项

### 通用注意事项
- 转换为 JPG 格式时，透明背景会被替换为白色
- 对于大型图片，转换可能需要一些时间

### Python 工具注意事项
- 确保 `webp` 文件夹存在且包含要转换的 WebP 文件
- 脚本会自动创建输出目录，无需手动创建

### 前端网页工具注意事项
- 由于浏览器限制，转换大型图片可能会导致性能问题
- 建议批量转换时不要选择过多文件
- 转换结果仅保存在浏览器内存中，刷新页面后会丢失，需及时下载

## 技术实现

### Python 工具
- 使用 Python 的 `os` 模块进行文件操作
- 使用 `Pillow` 库进行图片格式转换
- 命令行交互界面

### 前端网页工具
- 使用 HTML5 的 `File API` 读取本地文件
- 使用 `Canvas API` 进行图片格式转换
- 使用原生 JavaScript 处理用户交互
- 使用 CSS3 实现响应式布局

## 选择建议

- **Python 工具**：适合批量转换大量文件，或需要在脚本中集成转换功能的场景
- **前端网页工具**：适合快速转换少量文件，或在没有 Python 环境的情况下使用
